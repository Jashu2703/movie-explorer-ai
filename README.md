## 🧠 About the Project
This is a full-stack AI-powered movie recommendation web application that allows users to search for movies and get personalized recommendations based on content similarity.

The system uses machine learning techniques (TF-IDF + cosine similarity) on a dataset of 87K+ movies to generate recommendations in real time.

## ✨ Features
- 🔍 Movie search functionality
- 🎯 AI-based recommendations
- ⚡ Real-time results
- 📊 Content-based filtering
- 🌐 Deployed live web application

- ## 🛠️ Tech Stack
- Frontend: React + Vite
- Backend: Python
- ML: Scikit-learn, Pandas
- Deployment: NextToken Cloud

## Getting Started

### Backend Setup
1. Navigate to `backend/`
2. Create a virtual environment: `python -m venv venv`
3. Activate it: `source venv/bin/activate` (or `venv\Scripts\activate` on Windows)
4. Install dependencies: `pip install -r requirements.txt`
5. Run the server: `python main.py`

### Frontend Setup
1. Navigate to `frontend/`
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Build for production: `npm run build`

## Project Structure
- \`backend/\`: Python API and recommendation engine.
- \`frontend/\`: React + Vite + Tailwind CSS application.
- \`backend/data/\`: Contains the movie datasets (CSV).

## Technologies
- **Frontend**: React, Lucide-React, Tailwind CSS, Vite.
- **Backend**: Python, Pandas, Scikit-learn.
